## Running the project

This project uses pnpm as the package manager.

### Installation

```bash
# Install dependencies
pnpm install
```

### Development

```bash
# Start the development server
pnpm dev
```

### Production

```bash
# Build for production
pnpm build

# Preview the production build
pnpm preview
```
